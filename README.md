# MorphemeParser
* Data preprocessing code for [ChordingCoding](https://github.com/salt26/chordingcoding).
* Run with [Korean Sentiment Lexicon](http://word.snu.ac.kr/kosac/lexicon.php) for `Korean()`.
* Run with [Inquirer Dictionary](http://www.wjh.harvard.edu/~inquirer/homecat.htm) for `English()`.
